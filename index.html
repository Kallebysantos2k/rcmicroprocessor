Acerca do trabalho

Este trabalho foi realizado pelo aluno Kalleby Santos na disciplina de Redes e comunicações juntamente com a disciplina de Arquitetura de Computadores.

Este trabalho têm o intuito de fundamentar conteúdos acerca dos microprocessadores da disciplina de AC além de ser uma oportunidade para se praticar conteúdos do âmbito de Redes, tal como HTML 5, CSS e JavaScript. Para isso a turma foi dividida em diversos grupos onde cada um deveria abordar três tópicos especificos acerca dos microprocessadores.

O principal objetivo deste trabalho é ser uma página web onde possámos encontrar conteúdos acerca dos microprocessadores, de forma a aprender mais sobre os mesmos. Como toda a matéria foi dividida pelos grupos da turma 2ºK, o meu grupo ficou encarregado de abordar acerca:

  • Evolução dos microprocessadores
  • Execução de um programa residente em memória
  • Técnicas de processamento paralelo
